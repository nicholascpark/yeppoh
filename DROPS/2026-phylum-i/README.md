# Yeppoh: Emergent Phylum I

Target: 10 editions, Looking Glass + physical sculpture + NFT, $15-25K each.
Launch target: 2026-07-01.

## Folder structure

- `manifest.json` — drop-level metadata (date, counts, prices, contract address)
- `creatures/` — one subfolder per named creature (`NNN-<name>/`)
- `fabrication/` — STL exports, fabricator specs, quality notes
- `provenance/` — deployed contract snapshot, minting tx hashes, IPFS CIDs

## Permanence

Per `DIRECTION.md` §2, this folder is **permanent and additive**. Once the drop
launches, nothing here is modified, renamed, or removed. Corrections are made
forward — in the next drop, in a follow-up note, or in a new file — never by
editing what exists.

No force-push. No squash-merge that erases evolution. `git` history is sacred.
